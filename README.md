# 📍 مدنين لينك (Medenine Link)

منصة رقمية مبتكرة صممت خصيصاً لمساعدة وتسهيل وصول منتجات محلات ولاية مدنين (جنوب تونس) إلى المشترين في كافة ولايات الجمهورية التونسية عن طريق **"وسطاء تسوق محليين معتمدين"**.

## 🚀 المميزات الرئيسية للموقع:
- **دليل تفاعلي للمحلات:** تصنيف وعرض محلات الصناعات التقليدية، الملابس، والمنتجات الغذائية.
- **نظام طلب وسيط سريع:** يتيح للمشترين طلب "متسوق شخصي" يقوم بالمعاينة والشحن بالنيابة عنهم.
- **متوافق بالكامل مع الجوال:** تصميم عصري ومرن باستخدام Tailwind CSS.
